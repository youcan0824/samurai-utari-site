## ロゴシステム構成

### 01｜PRIMARY MARK（剣梅紋 リファイン版）─ `final_*`
**主用途**：名刺／Web／アプリアイコン／サイン／刻印／ファビコン／あらゆる基本シーン

五弁の切先に反り（sori）を導入し、家紋の気品と刀の鋭さを一記号に圧縮。花芯は凝縮された朱にヒラメキの閃光を内包。16pxでも識別可能な道具性と、1200px超の解像度でも耐える精緻性を両立。

- `final_mark.svg` / `.png`（透過）
- `final_mark_white.svg` / `.png`（和紙色背景）
- `final_mark_mono.svg` / `.png`（完全モノクロ）
- `final_en.svg` / `.png` / `final_en_white.*`（SAMURAI UTARI / INCORPORATED）
- `final_jp.svg` / `.png` / `final_jp_white.*`（株式会社サムライウタリ）
- `final_favicon_48.png` / `_32.png` / `_16.png`（favicon 専用）

### 02｜SECONDARY MARK（刀＋斬撃 / セレモニアル版）─ `mark_only.*`, `logo_en.*`, `logo_jp.*`
**主用途**：式典／大判ポスター／装幀パッケージ／ブランドムービーのタイトルカット

打刀と一閃の斬撃線を含む高解像度専用の華燭版。プライマリよりも情報量が多いため、細部が効く場面でのみ使用する。黄金比 φ=1.618 を全主要寸法に適用した緻密設計。

### 03｜SUB MARK（縮減版梅紋）─ `variant_A_*`
**主用途**：セキュリティ印／契約刻印／サブブランド／透かし／刺繍

最も控えめな梅紋のみ構成。プライマリのさらに簡略版として、補助的な用途でブランドの一貫性を保つ。モノクロ適性が高い。
